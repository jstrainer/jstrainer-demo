# jStrainer Demo

A simple demo of [jStrainer](https://github.com/jstrainer/jstrainer)

### Requirements

* Java 1.8 or later
* Maven 3

### How to run

```
git clone https://github.com/jstrainer/jstrainer-demo.git
cd jstrainer-demo
mvn spring-boot:run
```
